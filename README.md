# Accessibility Plugin

## Installation
```html
<!-- CDN -->
<script src="https://cdn.jsdelivr.net/gh/yourusername/accessibility-plugin@latest/dist/plugin.min.js"></script>

<!-- With configuration -->
<script>
window.A11Y_CONFIG = {
  buttonSize: '50px',
  position: 'bottom-left'
};
</script>
Options
Parameter	Default	Description
buttonSize	'40px'	Control button dimensions
position	'top-right'	Position on screen
